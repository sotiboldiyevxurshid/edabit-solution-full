## [39. Compare Strings by Count of Characters](https://edabit.com/challenge/yHGowWucg3k2kJdZ4)

**Answer**

```js
function comp(str1, str2) {
	return str1.length === str2.length ? true : false
}

comp("AB", "CD") ➞ true

comp("ABC", "DE") ➞ false

comp("hello", "edabit") ➞ false

```
