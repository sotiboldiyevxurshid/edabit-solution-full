## [1. Return the Sum of Two Numbers](https://edabit.com/challenge/3LpBLgNRyaHMvNb4j)

**Answer**

```js
function addition(a, b) {
	return a + b
}

addition(3, 2) ➞ 5

addition(-3, -6) ➞ -9

addition(7, 3) ➞ 10
```

## [2. Convert Minutes into Seconds](https://edabit.com/challenge/8q54MKnRrm89pSLmW)

**Answer**

```js
function convert(minutes) {
	return minutes * 60
}

convert(5) ➞ 300

convert(3) ➞ 180

convert(2) ➞ 120

```
